# Boston Housing Price Prediction

## Description
This Python script predicts the median value of owner-occupied homes in Boston using linear regression. It applies both gradient descent and the normal equations method for parameter estimation. 

## Requirements
- Python 3
- pandas
- numpy
- scikit-learn

## Usage
1. Make sure the `boston.csv` dataset is in the same directory as the script or adjust the `file_path` variable in the script accordingly.
2. Run the script using Python command: python BostonPricePrediction.py
3. Results will be saved to results.txt
